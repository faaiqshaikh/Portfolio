# Portfolio

## Overview

This repository contains a portfolio/resume landing page for Mohammedfaiq Shaikh. The project is implemented as a single-page HTML file that showcases services, experience, featured projects, skills, stats, education, and contact details.

## What is included

- A modern, dark-themed portfolio homepage built entirely in `index.html`
- Responsive, mobile-friendly layout for desktop and smaller screens
- Custom styles, animations, and interactions using HTML, CSS, and JavaScript

## Key features

- Hero section with styled name, description, and call-to-action
- Fixed navigation bar with smooth section links
- Custom cursor and hover ring for an interactive browsing experience
- Scroll progress bar at the top of the page
- Reveal animations for content as the user scrolls
- Parallax hero background and animated skill bars
- Service cards, experience timeline, featured project tiles, and skill tool list
- Stats section with delivery, projects, and client success indicators
- Contact and education information with a polished footer

## Technologies used

- HTML5
- CSS3 with custom layout styles and responsive design
- JavaScript for interactivity, scroll effects, and animations

## How to view

1. Open `index.html` in your browser
2. Scroll through the page to see the animated sections and portfolio content
3. Inspect the file to review the internal styles and script logic

## Notes

The HTML file is self-contained and includes all layout styles and JavaScript behavior directly inside the file. This makes it easy to preview without additional build tools or dependencies.
